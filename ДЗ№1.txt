ДЗ №1. HTML
Нужно сверстать 2 HTML страницы на тему "О себе". Не хотите о себе - придумайте любую другую тему.

Первая страница должна содержать:

Заголовок (элемент header, элементы h1 - h6)
Основную часть (элемент main, а в нём - элементы section, article, p и т.п.)
Футер (элемент footer, в нём какая-то информация)
Одно изображение
Ещё одно изображение - ссылка (по клику на него можно перейти на какую-то внешнюю веб-страницу, например, аккаунт в соц. сети)
Один обычный список с текстовыми пунктами (ul или ol) 
Один неупорядоченный список (ul) со ссылками для навигации (например, с пунктами "Homepage", "About me", "Contacts"). Один из этих пунктов должен быть 
ссылкой на вторую HTML страницу, остальные должны вести к якорям на текущей странице (по клику на них страница прокручивается к нужной секции)
Несколько абзацев текста, можно с заголовками
Таблица с произвольной информацией (желательно использовать где-то атрибут colspan или rowspan)
Любые другие элементы, рассмотренные на лекции. Чем больше разных элементов попробуете применить - тем лучше.
Вторая страница так же должна содержать структурные элементы (header, main, footer) и любое дополнительное содержание по вашему желанию. Главное, на ней 
должна располагаться форма для ввода данных с определённой целью (например, отправка контактов, регистрация пользователя, подписка на рассылку,
 заказ обратного звонка и т.д. и т.п.). Форма должна содержать, как минимум:

Текстовое поле ввода (input type="text")
Числовое поле ввода (input type="number")
Поле ввода электронного адреса (input type="email")
Поле для загрузки файла (input type="file")
Поле для многострочного текста (textarea)
Чекбокс
2 или более радио-кнопки
Выпадающий список (select)
Можете попробовать и другие поля ввода (input type="date", input type="range" и т.п.) Используйте различные атрибуты полей (required, disabled, и т.п.)
